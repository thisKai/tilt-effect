    Tilt Effect

    Windows Phone/Windows 10 style tilt effect for DOM elements written in javascript demo

    Prerequisites

    This library uses W3C Pointer Events, so you will need a polyfill — JQuery PEP works well.

    Basic Usage

    CommonJS

    var TiltEffect = require('tilt-effect')

TiltEffect({
  tiltElement: document.getElementById('tilt-this')
})

    ES2015

    import TiltEffect from 'tilt-effect'

TiltEffect({
  tiltElement: document.getElementById('tilt-this')
})
